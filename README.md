#### Installation Guide

> **NOTE**
>
> For Windows users, you must run all these installation scripts in `Git Bash`.

```bash
git clone https://github.com/OverbearingPearl/overbearing_gles
```

```bash
cd overbearing_gles/
```

```bash
git submodule sync --recursive && git submodule update --recursive --init
```

```bash
mkdir -p build/

```

```bash
cmake ..
```

```bash
make install
```
